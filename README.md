# 🚀 Mars Colonization Authority

A fully serious corporate simulation game about buying and selling Mars.

Built by Diya for Mansi. 💖

---

## How to play

Just open `index.html` in any browser — or host it on GitHub Pages.

## Deploy to GitHub Pages

1. Create a new GitHub repo (e.g. `mars-game`)
2. Upload all these files keeping the folder structure
3. Go to **Settings → Pages → Source → main branch / root**
4. Share the link with Mansi 🚀

## Tech

- HTML + CSS + Vanilla JS
- Phaser 3 (stars, planet, crying animation)
- Web Audio API (all sounds synthesized — no files needed)
- GitHub Pages (free hosting)
